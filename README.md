# Systems Tower Demo

Landing comercial independiente para Systems Tower.

## Ejecutar localmente

```powershell
py app.py
```

Abrir:

```text
http://localhost:5000
```

## Railway

Este proyecto puede desplegarse en un Railway separado usando:

- `app.py`
- `requirements.txt`
- `Procfile`
- `templates/index.html`

Railway debe detectar Python e instalar las dependencias de `requirements.txt`.
